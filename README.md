# Gulp sass+browser-sync
### Dependences
* Installed NPM [Node.js website](https://nodejs.org/ "Node.js") 
* Global installed Gulp-cli `npm install -g gulp-cli`
### Install dev dependencies
`npm install`
### Start dev server
`npm run serve` 
or 
`gulp`
### Build project
`npm run build` 
or 
`gulp build`
